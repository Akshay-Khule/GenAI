## Overview

AI Steward Connector Registry provides automated data discovery tools while leveraging AI agents for higher-level intelligence tasks. The solution bridges automated scanning with LLM-powered analysis for comprehensive data stewardship.

## Features

- **Postgres Auto-Discovery**: Automatically discover and catalog Postgres database assets
- **S3 Prefix Scan**: Efficiently scan and index S3 storage with prefix-based discovery
- **MCP Protocol Integration**: Expose discovery tools through standardized MCP interface
- **AI Agent Integration**: Delegate documentation, remediation, and analysis tasks to LLM-powered agents

## Architecture

The solution follows a layered approach:

1. **Discovery Layer**: Automated scanning of Postgres and S3 data sources
2. **Protocol Layer**: MCP-based tool exposure and standardization
3. **AI Layer**: LLM-powered agents for documentation and remediation tasks

## Use Cases

- Automated data cataloging and inventory
- Storage optimization and cleanup recommendations
- Compliance and governance reporting
- Data lineage and dependency mapping
- Intelligent data remediation planning
